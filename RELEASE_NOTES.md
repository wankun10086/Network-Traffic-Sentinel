# 流量哨兵 (Network Traffic Sentinel) 1.1.0 版本发布说明

流量哨兵 1.1.0 版本正式发布。本版本是一次全面的视觉系统重构与分析维度升级，引入了基于物理学阻尼的转移动画、高灵敏度的 IP 信誉审计模块，以及基于地理坐标投影的流光路由拓扑图。

---

## 1. 视觉系统重构：毛玻璃美学与空间微调

在 1.1.0 版本中，我们舍弃了传统的平面卡片分栏设计，转而采用以 Apple 设计原则为参考的毛玻璃（Glassmorphic）视觉体系：
* **磨砂背板滤镜**：信息卡片采用 `backdrop-filter: blur(30px) saturate(160%)` 混合半透明背景，能够自适应底层桌面或应用画布的色彩变化。
* **物理像素边界**：引入了极细的 Whisper Border（0.08 透明度边框），取代传统的强对比实线，在兼顾暗色与亮色主题下数据分割线清晰度的同时，避免了界面的凌乱感。
* **非对称空间结构**：放弃对称的等高布局，使用两栏非对称网格结构分配页面宽度，确保核心图表与数据流表格能够根据窗口宽度变化进行智能空间避让。

---

## 2. 动效重构：macOS 缓动物理动画

为降低界面数据频繁更新带来的视觉疲劳，我们重构了全部页面转移与组件拉伸的过渡机制：
* **阻尼滑动切换**：标签页切换（Tab Pane Transitions）引入了专为系统级偏好设置调校的 `tabFadeIn` 动效。该动画执行周期为 `350ms`，配合 `cubic-bezier(0.16, 1, 0.3, 1)` 缓动曲线，在淡出的同时伴有向上的微小位移，实现无震荡、平滑减速的视觉体验。
* **硬件加速渲染**：所有的动态过渡均绑定在 GPU 硬件加速的 `transform` 和 `opacity` 属性上，避免因 CPU 重绘引发的局部帧率波动，在多进程高负载采样时依然能稳定运行于 60fps+。

---

## 3. 分析升级：IP 物理属性标记

新版本在网络连接跟踪中引入了多源 IP 属性聚合，为所有外部套接字连接实时附加物理属性标记（IP Type Badges）：
* **标记分类**：
  * **住宅宽带 (Residential)**：基于住宅用户的常规连接。
  * **企业专线 (Business)**：专线及商业办公网络。
  * **数据中心 (Hosting)**：归属于云服务商、服务器机房或云主机集群。
  * **蜂窝数据 (Mobile)**：移动基站广播网络。
  * **CDN 边缘节点 (CDN)**：内容分发加速边缘节点。
  * **任播网络 (Anycast)**：基于 Anycast 路由的全局负载分发网络。
  * **未知网络 (Unknown)**：特征不明的网络节点。
* **置信度分级**：对每个标记划分 `exact`（完全精确匹配）与 `estimated`（智能网络估算）置信度标签，以便用户识别数据可信等级。

---

## 4. 路由轨迹剖析：太平洋中心拓扑图

为了解决传统地图在展示亚太与美洲网络包连线时从中间断裂的问题，我们开发了 Pacific-centered Transit Map：
* **地理坐标系重投影**：以经度 160° 为中心进行二次投影计算，本机节点与境外端点之间的路由连线呈平滑的抛物线几何路径。
* **渐变流光渲染**：连接链路内置了 `#flowGradient` 流体动画，流光轨迹沿物理网络传输方向循环流动，动态融合了上行（绿色）、下行（蓝色）与境外中继（橙/红色）的状态指示。
* **路由延迟时序**：跳转时序时间轴自动标识每一跳的延迟数据，将延迟大于 `150ms` 的跨洋段与少于 `40ms` 的局域段进行色彩警告区分，便于用户诊断跨国海缆瓶颈。

---

## 5. 小组件精简：三段式常驻悬浮窗

Tauri 独立的桌面小组件获得了性能与尺寸上的双重优化：
* **三级显示模式 (S / M / L)**：
  * **精简模式 (S)**：仅消耗个位数像素宽度，展示当前收发总速率。
  * **标准模式 (M)**：增加日累计用量与境外连接比率指标。
  * **详细模式 (L)**：支持直接渲染最活跃进程名称以及境外比率进度条。
* **背景穿透与置顶**：提供无边框拖拽，并支持在设置中动态调整背景透明度至 0.92，降低工作界面的视觉侵入感。
