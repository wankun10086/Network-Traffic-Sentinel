# 流量哨兵 (Network Traffic Sentinel) 用户指南

欢迎使用流量哨兵。本工具致力于为专业用户提供精准、低打扰、高可视性的本地网络流量监测与路由轨迹剖析服务。本指南将引导您熟悉界面结构、分析核心指标并调整本地悬浮小组件。

---

## 1. 界面设计风格与全局交互

流量哨兵采用深浅动态切换的毛玻璃（Glassmorphic）视觉体系。界面设计以数学精度与结构化排版为核心，通过以下交互逻辑提升数据呈现的清晰度：
* **动态背景晕光**：在深色背景下，窗口左上角与右下角分别渲染了超低饱和度的蓝色与绿色晕光（Radial Gradient），用于打破单调的背景色差，同时不干扰文字和图表的可读性。
* **磨砂玻璃卡片**：所有的功能面板与指标卡片均使用 `backdrop-filter: blur(30px) saturate(160%)` 处理，并在边缘辅以 1px 的超细半透明边界（Whisper Border），使内容区域在悬浮滚动时保持层级透视关系。
* **流控减速转移动画**：标签页切换及右侧抽屉式详情面板（IP Details Drawer）的弹出过程，均采用了符合苹果人机交互物理规律的缓动曲线（`cubic-bezier(0.16, 1, 0.3, 1)`）。标签切换伴随微小的垂直位移（`6px`）与渐隐渐现，确保视觉焦点的平滑过渡。

## 2. 导航标签页功能详解

主控制台通过顶部 Segmented Control 药丸滑块提供 6 个核心视图的切换：

### 2.1 总览 (Overview)
提供全局状况的动态快照。主要模块包括：
* **核心指标卡片**：展示今日累计上传/下载量、境外网络流量占比以及当前的网卡活跃数。
* **应用流量排行**：提取当前带宽消耗最高的进程，显示其对应的 PID、实时网速及累计用量。
* **最新告警列表**：展示触发异常规则的安全事件，如突发流量超限、未知 IP 扫描等。

### 2.2 应用 (Apps)
聚焦应用进程的多维度监控矩阵：
* **矩阵表格**：支持按进程名称或 PID 进行精准过滤。
* **监控字段**：可查看各进程的当前上传速率、当前下载速率、今日累计流量、历史连接总数以及境外通信比重，便于快速锁定高能耗进程。

### 2.3 连接 (Realtime)
提供底层的 TCP/UDP 实时会话流：
* **流数据展示**：每秒刷新当前的物理套接字连接，显示协议类型（TCP/UDP）、本地地址与端口、远端 IP 与端口、所属应用进程。
* **地址归属与置信度**：结合本地离线地理数据库，对每个 IP 进行大洲/国家分类，并标注“完全精确（exact）”或“估算（estimated）”的归属置信度。
* **详情下钻**：点击任意一行连接，可自右侧拉出“连接端点轨迹剖析”抽屉，进一步查询路由轨迹。

### 2.4 历史 (Stats)
长期流量账本与趋势分析视图：
* **时间粒度切换**：支持按日、周、月维度查看流量柱状图。
* **趋势图表**：使用 Tabular Numerals（等宽数字）对坐标轴数值进行排版，可单独隐藏或显示上传、下载或境外流量的曲线，帮助用户完成周期性用量对账。

### 2.5 地理/境外 (Overseas)
非中国大陆公网 IP 及未知归属的专项审查视图：
* **流量特征过滤**：隔离出所有目的地属于中国大陆以外的活跃连接。
* **分布热度排行**：按国家和地区统计连接分布密度，揭示境外数据传输的频率与累积大小。

### 2.6 规则与设置 (Control)
系统行为的定制中心：
* **采集偏好**：是否包含虚拟网卡数据、历史记录保留周期。
* **阈值告警设置**：支持单独开启日流量、月流量及境外流量配额预警。
* **悬浮窗全局控制**：开关悬浮窗、调整透明度等。

## 3. IP 纯净度审计 (IP Purity Audit)

当您在“连接”或“地理”视图中点击某个连接行时，右侧抽屉将展示对远端 IP 的完整纯净度审计结果：

### 3.1 评分区间 (Purity Score)
基于实时情报与网络拓扑属性，系统计算出一个 0 到 100 之间的信誉分数：
* **80 - 100 分（绿色）**：判定为干净安全（Clean）。通常是普通的住宅宽带或正规的企业专线 IP，无恶意扫描或代理特征。
* **40 - 79 分（黄色）**：判定为中低度威胁。通常归属于云数据中心（Hosting）、公开代理服务器或常见 VPN 出口，适合日常合规性检查。
* **0 - 39 分（红色）**：判定为高风险（Low Purity）。检测到公共代理（Proxy）、Tor 暗网出口（Onion Node）或在威胁情报中有恶意端口扫描与反弹控制记录，建议立刻切断连接。

### 3.2 四项核心审计项
每个被检测的 IP 都将通过以下校验过程，并在报告中标记“通过”或“检出”：
1. **数据中心/托管中转校验**：判定该 IP 是否由云服务商（AWS, Azure, Cloudflare 等）广播。若属数据中心，则会降低基准分。
2. **公共代理特征检测**：匹配已知的 HTTP/SOCKS5 公开代理指纹。
3. **洋葱网络/Tor 节点比对**：查询该 IP 是否列入活跃的 Tor Relay 列表中。
4. **威胁情报历史记录**：关联是否含有滥用行为或历史恶意报告记录。

## 4. 中转流光路由拓扑 (Transit Route Map)

路由轨迹剖析利用了 Pacific-centered（太平洋中心投影）的世界地图视图，直观展示该网络包从“本机”出发到达终点所经过的中间网络中继。

* **太平洋中心投影绘制**：不同于传统的欧洲中心地图，本投影将亚洲与美洲分居左右两侧，避免了跨太平洋网络包在绘制中线时出现断裂，更符合亚太用户的流量路径逻辑。
* **流光渐变路径**：路由链路使用三次贝塞尔曲线绘制，并通过 SVG 渐变通道渲染一条带方向的脉冲流光。流光颜色根据两端的状态（本地绿色、下载蓝色、境外黄色）进行平滑过渡。
* **路由跳转时序 Timeline**：在拓扑图下方，时序图列出每一跳（HOP）的路由节点，包含跳数、地理归属地、IP 物理地址、主机反查域名以及跳跃延迟。延迟标注为三个层级：
  * **绿色 (<40ms)**：本地与周边低延迟骨干网。
  * **橙色 (40-150ms)**：跨省、跨运营商中继。
  * **红色 (>150ms)**：跨洋海缆或国际路由节点。

## 5. 桌面悬浮小组件控制 (Local Widget)

小组件是常驻桌面的微型悬浮窗，专为在工作时不干扰主屏幕区域而设计。

### 5.1 三种布局模式
您可以通过小组件右上角的尺寸控制钮（S / M / L）即时切换布局样式：
* **小尺寸 (Small - S)**：仅显示当前实时的上传与下载带宽速率（例如：`24.3 MB/s` / `1.1 MB/s`）。
* **中尺寸 (Medium - M)**：除实时网速外，在下方增加今日流量累计字节数与当前的境外流量占比。
* **大尺寸 (Large - L)**：在中尺寸基础上，额外增加当前活动最高（Top Active）的进程名称，并显示一个以境外流量比重为基础的动态进度条。

### 5.2 操控特性
* **自由拖拽**：鼠标左键按住小组件顶部的句柄栏（Handle Bar）即可在多屏幕空间内任意移动。
* **穿透与钉固**：小组件默认开启窗口钉固（Pin），确保其不被其他全屏应用遮挡，可通过控制面板调整透明度使背景半透明化。
